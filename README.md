# T20 Match Winner Prediction
A Django-based web application that predicts the winner pf a T20 cricket match using a machine learning model.

## Tech Stack
- Python
- Django
- Scikit-learn
- Pandas
- NumPy

## Files
- randomforest_model.pkl : Trained ML model
- scaler.pkl : Feature scaler
- feature_selection1.pkl : Selected features
